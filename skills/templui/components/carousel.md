# Carousel

> Carousel component with navigation controls, autoplay, looping, and drag/swipe support.

## Install

```bash
templui add carousel
```

Dependencies auto-installed: icon

Add to your base layout `<head>`: `@carousel.Script()`

## Update

```bash
templui add carousel
```

To update all: `templui --installed add`

## API

### Enums

No enums.

### Props

```go
type Props struct {
	ID         string
	Class      string
	Attributes templ.Attributes
	Autoplay   bool
	Interval   int
	Loop       bool
}

type ContentProps struct {
	ID         string
	Class      string
	Attributes templ.Attributes
}

type ItemProps struct {
	ID         string
	Class      string
	Attributes templ.Attributes
}

type PreviousProps struct {
	ID         string
	Class      string
	Attributes templ.Attributes
}

type NextProps struct {
	ID         string
	Class      string
	Attributes templ.Attributes
}

type IndicatorsProps struct {
	ID         string
	Class      string
	Attributes templ.Attributes
	Count      int
}
```

## Components

| Function | Props | Description |
| -------- | ----- | ----------- |
| `Carousel(props ...Props)` | `Props` | Root wrapper `<div>`. Sets up carousel state via data attributes. `Autoplay` enables automatic slide advancement, `Interval` sets the delay in milliseconds (default 5000), `Loop` enables infinite cycling. |
| `Content(props ...ContentProps)` | `ContentProps` | Sliding track `<div>` that holds all items. Renders as a flex row with CSS transition for slide animation. Supports mouse drag and touch swipe. |
| `Item(props ...ItemProps)` | `ItemProps` | Individual slide `<div>`. Each item takes the full width and height of the carousel. Place inside `Content`. |
| `Previous(props ...PreviousProps)` | `PreviousProps` | Previous-slide `<button>` positioned at the left center. Renders a `ChevronLeft` icon. Automatically disabled at the first slide when `Loop` is false. |
| `Next(props ...NextProps)` | `NextProps` | Next-slide `<button>` positioned at the right center. Renders a `ChevronRight` icon. Automatically disabled at the last slide when `Loop` is false. |
| `Indicators(props ...IndicatorsProps)` | `IndicatorsProps` | Row of dot `<button>` elements centered at the bottom. `Count` is **required** and must match the number of `Item` children. The active indicator is highlighted automatically. |
| `Script()` | — | Outputs the `<script>` tag for carousel behavior (navigation, autoplay, drag/swipe). Auto-generated by `templui add`. Place in `<head>`. |

## Composition

```
Carousel
  Content
    Item
    Item
    ...
  Previous
  Next
  Indicators
```

`Content` wraps all `Item` slides as a horizontal track. `Previous`, `Next`, and `Indicators` are siblings of `Content` inside `Carousel`. `Indicators.Count` must equal the number of `Item` children.

## Dependencies

- `icon` — used for `ChevronLeft` and `ChevronRight` icons in navigation buttons.

## Context

No context passing.

## Data Attributes

| Attribute | Element | Description |
| --------- | ------- | ----------- |
| `data-tui-carousel` | `Carousel` root | Marks the root carousel container. |
| `data-tui-carousel-current` | `Carousel` root | Tracks the currently active slide index (zero-based). |
| `data-tui-carousel-autoplay` | `Carousel` root | `"true"` or `"false"` — whether autoplay is enabled. |
| `data-tui-carousel-interval` | `Carousel` root | Autoplay interval in milliseconds. Defaults to `"5000"`. |
| `data-tui-carousel-loop` | `Carousel` root | `"true"` or `"false"` — whether the carousel loops infinitely. |
| `data-tui-carousel-initialized` | `Carousel` root | Set to `"true"` by JS after first initialization. |
| `data-tui-carousel-track` | `Content` | Marks the sliding track container. |
| `data-tui-carousel-item` | `Item` | Marks an individual slide. |
| `data-tui-carousel-prev` | `Previous` | Marks the previous-slide button. |
| `data-tui-carousel-next` | `Next` | Marks the next-slide button. |
| `data-tui-carousel-indicator` | Indicator button | Zero-based slide index this indicator corresponds to. |
| `data-tui-carousel-active` | Indicator button | `"true"` or `"false"` — whether this indicator is the active one. |

## Usage

```templ
@carousel.Carousel(carousel.Props{Autoplay: true, Interval: 4000, Loop: true}) {
  @carousel.Content() {
    @carousel.Item() {
      <img src="/images/slide-1.jpg" alt="Slide 1" class="w-full h-64 object-cover" />
    }
    @carousel.Item() {
      <img src="/images/slide-2.jpg" alt="Slide 2" class="w-full h-64 object-cover" />
    }
    @carousel.Item() {
      <img src="/images/slide-3.jpg" alt="Slide 3" class="w-full h-64 object-cover" />
    }
  }
  @carousel.Previous()
  @carousel.Next()
  @carousel.Indicators(carousel.IndicatorsProps{Count: 3})
}
```
