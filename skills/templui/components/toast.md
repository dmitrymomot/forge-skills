# Toast

> Displays a temporary notification message with auto-dismiss, progress indicator, and configurable position.

## Install

```bash
templui add toast
```

Dependencies auto-installed: icon

Add to your base layout `<head>`: `@toast.Script()`

## Update

```bash
templui add toast
```

To update all: `templui --installed add`

## API

### Enums

```go
type Variant string

const (
	VariantDefault Variant = "default"
	VariantSuccess Variant = "success"
	VariantError   Variant = "error"
	VariantWarning Variant = "warning"
	VariantInfo    Variant = "info"
)

type Position string

const (
	PositionTopRight     Position = "top-right"
	PositionTopLeft      Position = "top-left"
	PositionTopCenter    Position = "top-center"
	PositionBottomRight  Position = "bottom-right"
	PositionBottomLeft   Position = "bottom-left"
	PositionBottomCenter Position = "bottom-center"
)
```

### Props

```go
type Props struct {
	ID            string
	Class         string
	Attributes    templ.Attributes
	Title         string
	Description   string
	Variant       Variant
	Position      Position
	Duration      int
	Dismissible   bool
	ShowIndicator bool
	Icon          bool
}
```

## Components

| Function | Props | Description |
| -------- | ----- | ----------- |
| `Toast(props ...Props)` | `Props` | Renders a fixed-position toast notification. Auto-generates an ID if none provided. Defaults: `Variant` = `VariantDefault`, `Position` = `PositionBottomRight`, `Duration` = `3000` (ms). |
| `Script()` | -- | Outputs the `<script>` tag for toast auto-dismiss, progress animation, and hover-pause behavior. Auto-generated by `templui add`. Place in `<head>`. |

## Composition

Flat component -- no sub-components.

The toast internally renders:
- An optional progress indicator bar (when `ShowIndicator` is `true` and `Duration` > 0).
- An optional variant icon (when `Icon` is `true`): `CircleCheck` for success, `CircleX` for error, `TriangleAlert` for warning, `Info` for info.
- Title and Description text from props.
- An optional dismiss button (when `Dismissible` is `true`), rendered as a ghost icon button with an X icon.

## Dependencies

- `icon` -- used for variant icons and dismiss button icon.
- `button` -- used for the dismiss button.
- `utils` -- used for `TwMerge`, `RandomID`, and `ScriptURL`.

## Context

No context passing.

## Data Attributes

| Attribute | Element | Description |
| --------- | ------- | ----------- |
| `data-tui-toast` | Root `<div>` | Marks the toast container. Used by JavaScript to initialize auto-dismiss and hover-pause behavior. |
| `data-tui-toast-duration` | Root `<div>` | Duration in milliseconds before auto-dismiss. |
| `data-position` | Root `<div>` | Position value (e.g. `"bottom-right"`). Drives CSS positioning via `data-[position=*]` selectors. |
| `data-variant` | Root `<div>`, progress bar | Variant value (e.g. `"success"`). Drives variant-specific color styling. |
| `data-duration` | Progress bar | Duration in milliseconds for the progress animation. |
| `data-tui-toast-dismiss` | Dismiss button | Marks the dismiss button. Clicked to close the toast. |

## Usage

```templ
import "github.com/templui/templui/components/toast"

@toast.Toast(toast.Props{
	Title:         "Changes saved",
	Description:   "Your profile has been updated successfully.",
	Variant:       toast.VariantSuccess,
	Position:      toast.PositionBottomRight,
	Duration:      5000,
	Dismissible:   true,
	ShowIndicator: true,
	Icon:          true,
})
```
